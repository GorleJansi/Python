## 6ï¸âƒ£ Iterables & Iterators

An **iterable** is an object that can be looped (list, tuple, string).  
An **iterator** gives one value at a time using `next()`.

```python

it = iter(s)
print(next(it))  # H
print(next(it))  # i
```
